What I learnt

  1. Documentation is important!
  2. Documentation is taught in USA universities, but not taught in UK or Singapore universities
      3. https://www.slideshare.net/sangi/what-is-documentation-and-its-techniques
  3. How to make Sublime Text 3 autofill a HTML template
    1. set the file type to "html" using the file extension or set it at the bottom right corner
    2. type "html"
    3. press tab to autocomplete
  4. Sublime Text has INFINITE UNDOS!
  5. Sublime Text: By typing the tag name (e.g "li") and pressing tab, you can autofill the whole tag
  6. Unless you want to be a web designer, you don't need to learn to be an expert in HTML
  7. File priorities when accessing the default page in a Github Pages page.
    • (highest priority first): index.html, index.md, readme.md
  8. Markdown can include html+css because it is ultimately converted to it
